send 2 ma mail is a wordpress plugin that have one mission only, improve the readability for your website's content, all by using an email input to the top of the content.

== Description ==
how many times did you found an interesting articles poping up on your facebook / twitter feed and you didnt had the time to read it?!

yes, i know you can use the faebook save button or other ways to save the ontent somewhere. but, what if the website will offer you to send yourself the article (or any other content) to your email address and you will be able to read it later... isnt it useful?

An other great use for this plugin is that website manager / marketers are asking for email address from visitors and offer them to join the mailing list using an interesting article that the visitor already like!



== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the tools->Send 2 ma mail screen to configure the plugin
